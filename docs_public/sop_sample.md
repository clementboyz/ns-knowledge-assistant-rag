# SOP (Sample): Equipment Borrow / Return

## Borrow
1. Record borrower name and ID.
2. Record item ID and condition.
3. Borrower signs digitally.
4. Admin verifies and approves.

## Return
1. Check item condition vs initial record.
2. Update return date/time.
3. Admin closes the case.

## Notes
- If item is damaged, create an incident record and inform the duty officer.
