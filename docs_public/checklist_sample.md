# Checklist (Sample): Borrow / Return Workflow

## Borrow Checklist (Admin)
- [ ] Confirm borrower identity (name + ID)
- [ ] Confirm borrower is authorized to borrow
- [ ] Record item ID
- [ ] Record item condition (remarks if any defects)
- [ ] Confirm accessories (if applicable): charger / cable / bag / etc.
- [ ] Borrower signs (digital / acknowledgement)
- [ ] Approve and issue item
- [ ] (Optional) Take a reference photo for condition proof

## Return Checklist (Admin)
- [ ] Confirm item ID matches the borrow record
- [ ] Check item condition vs initial record
- [ ] Confirm accessories returned (if applicable)
- [ ] Record return date/time
- [ ] Close the case / mark as returned

## Incident Checklist (if damaged / missing)
- [ ] Mark item as damaged/missing (do not re-issue)
- [ ] Create incident record
- [ ] Capture basic details:
  - what happened (short description)
  - time/date discovered
  - borrower/handler involved
- [ ] Inform duty/admin point-of-contact
- [ ] Arrange inspection / follow-up actions
